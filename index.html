Hur fungerar Privatkopiera?

Tillägget tar reda på vilka addresser du kan använda för att ladda ned videoklipp från SVT Play. Tillägget är inspirerat av Pirateplay. Webbappen på Pirateplay-sidan var trasig under en period, så jag bestämde mig för att göra en version som var enklare att använda.

Privatkopiera kräver att du vet hur man använder ffmpeg. Det är egentligen väldigt enkelt. Följ dessa steg:

  1. Hitta en video på svtplay.se som du vill ladda ned.
  2. Öppna Privatkopiera genom att klicka på VHS-ikonen till höger om addressfältet.
  3. Vänta några sekunder tills tillägget hittar strömmen.
  4. Klicka på knappen Kopiera kommando.
  5. Klistra in kommandot i en terminal (t.ex. cmd.exe i Windows). Tryck enter.
  6. Förhoppningsvis har du filen nedladdad om någon minut.

Om du tycker om tillägget, lämna gärna en recension i Chrome-butiken. Tack!

Installera ffmpeg

I Debian och Ubuntu:

{% highlight bash %} sudo apt-get install ffmpeg {% endhighlight %}

I Mac OS X (med Homebrew):

{% highlight bash %} brew install ffmpeg {% endhighlight %}

Windows är som vanligt krångligast. Ladda ned ffmpeg. Konfigurera så att ffmpeg.exe kan hittas i PATH.
